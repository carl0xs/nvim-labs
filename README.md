## Configs do neovim

Este repo centraliza diferentes abordagens e experimentos de configuracao do neovim que uso/usava no meu dia-a-dia.

_setup.sh_ é apenas um utilitario para clonar este repositório para '.config/' e definir funcões de shell como:

**nv** - para escolher qual configuracao quero usar. 


    1 - Lua
    2 - Vimscript
    3 - Nvim-packer - Packer manager nativo neovim
    3 - [Purevim](https://github.com/carl0xs/purevim.git)
    4 - [Nixvim](https://github.com/carl0xs/nixcfg/modules/workstation/neovim.nix)
